Tutorials:
This is the best place to show you how to get started!

1. Pick an HTML editor – choose the program for writing and editing your website’s code, like VSCode.
2. Plan the site layout – make a mockup of the website’s layout using a program like Adobe XD or a pen and paper.
3. Write the HTML code – create an index.html file and add HTML document structure tags.
4. Create the layout elements – separate your website into multiple sections based on the layout by adding tags to the HTML file.
5. Add the HTML content – add the heading title, body text, and images into each section.
6. Include layout CSS – create a style.css file and add CSS code to change the column position, text alignment, and elements padding.
7. Customize your site – use CSS style attributes to customize the website’s background color, font size, typefaces, and other visual elements.
